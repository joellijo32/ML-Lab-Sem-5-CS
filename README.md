# ML-Lab-Sem-5-CS

Machine Learning Lab coursework for Semester 5 (CS). This repository contains lab cycle solutions, some reference materials for Python data-science libraries, and solutions for the lab cycle questions.
            
## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

1. Clone the repository:
   ```bash 
   git clone https://github.com/joellijo32/ML-Lab-Sem-5-CS.git
   cd ML-Lab-Sem-5-CS
   ```

2. (Optional) Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate   # On Windows: venv\Scripts\activate
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Dependencies

- [NumPy](https://numpy.org/)
- [Pandas](https://pandas.pydata.org/)
- [Matplotlib](https://matplotlib.org/)
- [scikit-learn](https://scikit-learn.org/)
- [TensorFlow](https://www.tensorflow.org/)

## Usage

Run any solution or reference script directly with Python, for example:

```bash
python "Lab Cycle Solutions/1_LINEAR_REGRESSION/Q1_a_Linear_Regression_Gradient_Descent.py"
```

Jupyter notebooks under `references/Matplotlib/` can be opened with Jupyter Notebook/Lab or VS Code:

```bash
jupyter notebook "references/Matplotlib/Matplotlib.ipynb"
```
